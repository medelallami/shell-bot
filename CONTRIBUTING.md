# Contributing to shell-bot

We welcome contributions to shell-bot! Here are a few ways you can help:

## Reporting Bugs
If you find a bug, please open an issue with a clear description and steps to reproduce.

## Suggesting Features
Have an idea to make shell-bot better? Open an issue to discuss it!

## Submitting Pull Requests
1.  Fork the repository.
2.  Create a new branch for your feature or fix.
3.  Ensure your code follows the modern ES6+ standards used in the project.
4.  Submit a pull request with a clear description of your changes.

## Coding Standards
- Use `const` and `let` instead of `var`.
- Use `class` syntax for core modules.
- Use arrow functions where appropriate.
- Use `async/await` for asynchronous operations.
- Use template literals for string concatenation.

Thank you for contributing!
